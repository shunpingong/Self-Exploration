# Exploration Workspace

This repository is the parent workspace for experiments, prototypes, and local integrations. Each exploration lives in its own folder with its own README, code, configuration, and supporting files.

## Current Projects

- `Nemotron/` contains the FastAPI backend exploration for NVIDIA Nemotron.
- `Openclaw/` contains the Openclaw exploration for local LLM inference.

## Working Style

- Keep project-specific setup, notes, and dependencies inside each project folder.
- Add new explorations as sibling folders at the repository root.
- Use this root README for high-level navigation and context only.

## Adding More Explorations

Create a new top-level folder for each project, add a local `README.md`, and keep that folder self-contained. That keeps the repository organized even with more experiments over time.
